Project Overview
This assignment implements two advanced search systems:

 1. Hybrid Search for NIPS Papers
Web Scraping: Automated collection of NIPS 2024 conference papers

MongoDB Storage: Efficient document storage with authors as arrays

Intelligent Search: Combines semantic + keyword search

Features: Title, author, and keyword-based paper discovery

2. Reverse Image Search
Vision AI: Uses Vision Transformers (ViT) for image embeddings

Similarity Matching: Finds visually similar images using cosine similarity

Privacy-First: Local processing - no data sent to third parties

Mobile Ready: Can be converted to mobile app
